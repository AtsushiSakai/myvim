# Install

1　Checkout software

> git clone https://gist.github.com/8822390.git ~

2 Create symboric links

> ln -s ~/myvim/.vimrc ~/.vimrc ~/myvim/.vim ~/.vim

3 Install neobundle

> git clone git://github.com/Shougo/neobundle.vim ~/.vim/bundle/neobundle.vim
